# POST routes
* **POST** /user/register
	- `fullname`: fullname (string) 
	- `username`: username (string) 
	- `email`: email (string) 
	- `password`: password (string) 

* **POST** /user/login
	- `username`: username (string) 
	- `password`: password (string) 

* **POST** /user/gauth
	- `token`: token (string)
